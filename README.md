# Cinematic 3D Portfolio

A high-performance, immersive 3D portfolio website designed to showcase creative development skills. Built with **React**, **Three.js**, and **GSAP**, featuring a luxury monochrome aesthetic and interactive 3D elements.

## Features

### Cinematic Visuals
- **Monochrome & Gold Theme**: A premium "Dark Luxury" aesthetic using deep black (`#050505`) and metallic gold (`#D4AF37`).
- **3D Hero Section**: Features a randomized "hacker-style" name decryption animation with a permanent neon glow.
- **Atmospheric Effects**: Floating particles, cinematic bloom, film grain, and vignette post-processing.

### Skill DNA
- **Interactive Double Helix**: A mathematically accurate 3D DNA visualization representing technical skills.
- **Dynamic Nodes**: Gold spheres for Frontend skills, Silver for Backend/Creative.
- **Glassmorphism**: Detailed skill information displayed in a frosted glass panel on click.

### Core Architecture
- **Smooth Scroll**: Implemented using `Lenis` for a fluid, premium browsing experience.
- **Custom Cursor**: A magnetic cursor with a trailing follower (default system cursor hidden).
- **AI Widget**: A floating "Ask Me Anything" chat widget that simulates AI responses about the portfolio owner.

## Tech Stack

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **3D Engine**: [Three.js](https://threejs.org/) + [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber)
- **Animations**: [GSAP](https://gsap.com/) + [@gsap/react](https://gsap.com/resources/react/)
- **Post-Processing**: [@react-three/postprocessing](https://docs.pmnd.rs/react-postprocessing)
- **Styling**: Vanilla CSS (Variables & Glassmorphism)

## Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/saadaslam99/Muhammad-Saad-Portfolio.git
    cd Muhammad-Saad-Portfolio
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
